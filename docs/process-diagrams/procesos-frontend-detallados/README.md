# Procesos Frontend Detallados

Esta carpeta contiene la documentación detallada de los procesos principales del frontend de Discipline Rift, organizados con diagramas Mermaid.

## Archivos Incluidos

### 07-formulario-contacto.md
Documentación completa del proceso de envío del formulario de contacto, incluyendo:
- Flujo principal de envío
- Validación anti-spam (BotID + Honeypot)
- Secuencia de envío de email
- Validación de campos
- Proceso de sanitización

### 08-aplicacion-trabajo.md
Documentación completa del proceso de aplicación de trabajo para coaches, incluyendo:
- Flujo principal de aplicación
- Secuencia completa con subida de PDF
- Validación de PDF
- Proceso de subida con rollback
- Validación de campos
- Diagrama de estados

### 09-navegacion-ui.md
Documentación completa de los procesos de navegación y UI/UX, incluyendo:
- Sistema de navegación principal
- Estados del header dinámico
- Navegación con scroll a sección
- Cálculo de offset dinámico
- Navegación móvil
- Scroll automático al cargar página
- Estructura de rutas y secciones
- Sistema de componentes UI
- Manejo de mensajes de éxito
- Responsive design

## Uso

Cada archivo contiene múltiples diagramas Mermaid que pueden ser visualizados en:
- GitHub (renderizado automático)
- Editores Markdown con soporte Mermaid (VS Code con extensión)
- Herramientas online como mermaid.live

## Estructura

Todos los diagramas están organizados por función y tipo:
- **Diagramas de Flujo**: Procesos paso a paso
- **Diagramas de Secuencia**: Interacciones entre componentes
- **Diagramas de Estados**: Estados y transiciones
- **Diagramas de Decisión**: Validaciones y decisiones críticas
- **Diagramas de Estructura**: Arquitectura y organización
